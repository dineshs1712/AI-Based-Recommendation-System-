🤖 AI-Based Recommendation System

📌 Overview
This project is a simple AI-based recommendation system built using Python. It provides suggestions for movies, food, and games based on user input.
It is designed for beginners to understand how recommendation systems work using basic logic and data structures.

🚀 Features
Recommend Movies 🎬
Suggest Food 🍔
Suggest Games 🎮
User-friendly CLI interface
Beginner-friendly project

🧠 Technologies Used
Python
Basic Data Structures (Dictionary, List)
Conditional Statements

📂 Project Structure

recommendation-system/
│
├── main.py        # Main program
└── README.md      # Documentation

⚙️ How It Works
User selects a category (Movies / Food / Games)
User enters preference (like genre/type)
System fetches data from predefined dictionary
Displays recommendations

🧪 Sample Run

===== AI Recommendation System =====
1. Movies
2. Food
3. Games

Select option: 1
Enter genre (action/comedy/horror): action

Recommended Movies:
- Avengers
- Batman

▶️ How to Run
Bash
# Run the Python file
python main.py
